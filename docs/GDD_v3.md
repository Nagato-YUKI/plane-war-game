# 雷电战机 - 游戏设计文档 V3

## 1. 核心玩法改进

### 1.1 弹幕躲避为核心
- 敌人弹幕密度大幅增加，形成真正的"弹幕地狱"
- 玩家需要专注于躲避而非主动攻击
- 引入"擦弹"机制：子弹靠近玩家但不命中时获得额外分数
- 玩家有小型碰撞体积（35%），便于在弹幕缝隙中穿梭

### 1.2 武器道具持续时间机制
- 所有武器道具都有持续时间，不是永久升级
- 武器等级会随时间衰减，鼓励玩家持续收集道具
- 武器持续时间：15-30秒
- 武器等级越高，持续时间越短（平衡性）

### 1.3 敌人数量和密度
- 基础敌人数量：8-25个（根据波次和难度）
- 敌人刷新间隔：200-800ms
- 引入"敌人群体"概念：同类型敌人一起出现形成弹幕阵

## 2. 视觉表现改进

### 2.1 AI生成素材
- 玩家战机：5种主题皮肤（雷电、烈焰、冰霜、暗影、圣光）
- 敌人：4种基础敌人 + 2种Boss
- 子弹：不同颜色和大小的弹幕
- 特效：爆炸、引擎尾焰、护盾、擦弹特效

### 2.2 特效系统
- 引擎尾焰粒子效果
- 子弹拖尾效果
- 爆炸粒子效果
- 擦弹时的闪光效果
- 屏幕震动效果

## 3. 游戏机制

### 3.1 连击系统
- 击杀敌人增加连击数
- 擦弹增加少量连击
- 连击随时间衰减
- 连击倍率：1x-5x

### 3.2 成就系统
- 初次出击、弹幕大师、连击之王
- 无伤通关、弹幕地狱（疯狂难度通关）

### 3.3 难度系统
- 简单、普通、困难、疯狂
- 影响敌人HP、弹幕速度、密度、分数倍率

### 3.4 Boss战
- 每5波出现Boss
- Boss有多阶段，HP降低时切换更密集的弹幕模式
- 机械巨兽和生化母舰两种Boss

## 4. 解锁系统
- 累计分数解锁新皮肤和武器
- 皮肤：雷电（默认）、烈焰（500分）、冰霜（1500分）、暗影（3000分）、圣光（5000分）
- 武器：火神炮（默认）、激光（300分）、导弹（800分）

## 5. 操作方式
- 键盘方向键移动，空格射击
- 鼠标移动控制（平滑移动）
- 触屏移动控制
- X键释放炸弹

## 6. 新增改进方案

### 6.1 武器持续时间可视化
- 武器等级条显示在玩家战机上方
- 颜色从绿到红表示剩余时间
- 即将失效时闪烁提示

### 6.2 弹幕预警系统
- Boss释放大型弹幕前显示预警区域
- 红色闪烁提示危险区域

### 6.3 敌机入场动画
- 敌人从屏幕外飞入，有入场轨迹
- 增加游戏动感

### 6.4 动态背景
- 星空背景滚动
- 不同波次背景颜色微调

### 6.5 音效反馈
- 射击音效
- 爆炸音效
- 擦弹音效
- 道具拾取音效
