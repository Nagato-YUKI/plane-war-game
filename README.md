# 飞机大战

网页版飞机大战游戏 - 课程作业项目

## 游戏操作

- **方向键** ↑↓←→：控制飞机移动
- **鼠标拖动**：控制飞机移动 + 自动射击
- **空格键**：射击
- **触摸**：移动端触摸控制 + 自动射击

## 游戏规则

- 击毁敌机获得分数（+10分/架）
- 被敌机或敌机子弹击中则游戏结束
- 支持本地最高分记录
- 难度随时间递增（敌机速度、数量、射击频率逐步提升）

## 视觉特效

- Retro-Futurism / Cyberpunk 风格
- 霓虹发光效果（青色/品红）
- 滚动星空 + 星云背景
- 粒子系统（引擎尾焰、子弹拖尾、爆炸）
- 屏幕震动反馈
- CRT 扫描线效果
- 分数弹出动画
