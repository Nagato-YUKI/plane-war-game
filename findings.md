# 研究发现

## 现有功能清单

### 已实现
1. 三种游戏状态（开始/进行/结束）
2. 玩家飞机控制（键盘方向键 + 鼠标拖动）
3. 子弹系统（空格发射，200ms间隔）
4. 敌机系统（1秒间隔生成，最多8架，30%可射击）
5. 碰撞检测（AABB）
6. 分数系统 + localStorage 最高分
7. 图片降级绘制模式
8. 响应式 CSS 适配

### 缺失/问题
1. **触摸事件未实现** - handleTouchMove/handleTouchEnd 只有 TODO
2. **鼠标控制时无法射击** - 鼠标模式下没有自动射击机制
3. **缺少图片资源** - enemy.png, background.png, bullet.png 不存在
4. **无爆炸特效** - 敌机被击毁时无视觉反馈
5. **背景静态** - 没有滚动星空效果
6. **无粒子系统** - 缺少引擎尾焰等粒子效果
7. **无难度递进** - 游戏难度始终不变
8. **无生命值系统** - 一击即死，无容错
9. **无道具系统** - 缺少增益道具

## UI/UX 改进方向

### 视觉风格：Cyberpunk / Retro-Futurism
- 霓虹色彩（青色、品红、黄色）
- 发光效果和光晕
- 粒子特效（星空、尾焰、爆炸）
- 扫描线和 CRT 效果（可选）

### 关键特效
1. 滚动星空背景
2. 玩家飞机引擎尾焰粒子
3. 子弹拖尾效果
4. 爆炸粒子动画
5. 屏幕震动效果
6. 分数弹出动画
